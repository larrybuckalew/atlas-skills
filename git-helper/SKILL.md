# Git Helper

Common git operations as a skill. Provides convenient wrappers for frequently used git commands including status, pull, push, branch management, and log viewing.

## Commands

### git-helper status
Show working tree status - displays modified files, staged changes, and current branch.

### git-helper pull
Pull latest changes from the remote repository.

### git-helper push
Push local commits to the remote repository.

### git-helper branch
List or manage branches. Shows current branch and all local/remote branches.

### git-helper log [--limit 10]
View commit log with optional limit parameter. Default shows last 10 commits.

## Install

No installation needed. git is always present on the system.

## Usage

This skill provides quick access to common git operations for the current workspace repository.
